## README: Performance Management System (PMS)

This repository contains the source code for the **Digital Employee Performance Management System (PMS)**, a high-fidelity platform designed for **Dart Global Logistics Inc.**The system is engineered to replace manual performance tracking with an automated, AI-driven workflow that supports a global organizational structure

---

## 🚀 Project Overview

The PMS facilitates standardized performance evaluations across all global branches of DGL Inc., ensuring transparency and alignment with corporate goals. It features a comprehensive suite of tools for objective setting, year-round tracking, and advanced analytics.


## 🛠 Tech Stack

* **Frontend:** Next.js (TypeScript) with **Turbopack** for optimized build performance
* **Backend:** Python Flask API
* **Database:** **Supabase (PostgreSQL)** with Row-Level Security (RLS) for multi-tenant data isolation
* **Authentication:** Microsoft Active Directory SSO with MFA integration. (Only for final implementation)
