# PMS - Performance Management System

This is a full-stack **Performance Management System (PMS)** project.

It includes:
- Frontend (Next.js)
- Backend (Flask / Node depending on setup)
- Feature-based development using Git branches

---

## 🚀 Project Purpose

The PMS system is designed to manage and track performance-related data in an organized and scalable way.

---

## 🛠️ Tech Stack

- Next.js (Frontend)
- Node.js / Flask (Backend)
- Git & GitHub (Version Control)
